# Portafolio Personal

Este proyecto corresponde a una evaluación práctica del módulo de Front-End.

## Descripción
Consiste en la creación de un portafolio personal de una sola página, desarrollado con HTML5 y CSS3. El sitio incluye barra de navegación, sección sobre mí, habilidades, proyectos, formulario de contacto y footer.

## Tecnologías utilizadas
- HTML5
- CSS3
- Git
- GitHub

## Cómo ejecutar
1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en el navegador.
